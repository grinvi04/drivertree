## 1. 검증 가설 4개

PRD 1.4와 NEXT_STEPS.md 1.2에서 도출. 각 가설은 (1) 진술, (2) 검증 방법, (3) Pass/Fail 판정 기준, (4) 결과별 의사결정 분기 형식으로 정리.

### H1. 리텐션 — 얇은 시장 가설

**가설 진술**
> 초보운전자는 면허 취득 후 **1~2년 이내**에 *"이제 초보 졸업"* 으로 자기 정의하며 본 서비스를 졸업한다. 따라서 신규 유입이 끊기면 트래픽이 빠르게 빠진다.

**검증 방법**
- 인터뷰 질문 P1-Q5, P2-Q5/Q6, P3-Q5에서 본인의 *"초보 졸업 시점"* 인식을 파악.
- 졸업 후 재방문 의향을 추가로 질의.

**Pass 기준 (가설 확정)**
- 5명 중 **3명 이상**이 "졸업까지 1~2년" 답변
- AND 5명 중 **3명 이상**이 "졸업 후엔 거의 안 들어올 것 같다" 답변

**Fail 기준 (가설 반증)**
- "꽤 오래 초보라고 느낄 것 같다(3년+)" 또는 "필요할 때마다 다시 들어올 것 같다" 답변이 다수

**결과별 의사결정 분기**
- **Pass** → 신규 유입(SEO, 검색광고)이 사업의 핵심 lever. KPI는 MAU 성장률. 광고 모델로 충분.
- **Fail** → 기존 사용자의 LTV가 더 길다 → 차량 관리·구매·보험 등 인접 영역 확장 가능 → 구독 모델 고려 여지.

---

### H2. 재방문 동기 — 일회성 트래픽 가설

**가설 진술**
> 한 번 답을 얻고 나면 같은 사용자가 다시 들어올 동기가 약하다. **재방문 동기 설계가 없으면 일회성 트래픽으로 끝난다.**

**검증 방법**
- P1-Q6, P2-Q7, P3-Q6에서 재방문 후보 기능(커뮤니티 / 정비 알림 / 계산기 즐겨찾기 / 매월 새 콘텐츠 알림 / Q&A)을 제시하고 끌림 강도 측정.
- *"매주/매월 반복적으로 운전 관련해서 궁금한 게 있나"* 자유 답변 병행.

**Pass 기준 (가설 확정)**
- 5명 중 **3명 이상**이 "재방문 동기 없다" 답변
- OR 제시한 후보 중 강하게 끌리는 것이 5명 중 2명 이하

**Fail 기준 (가설 반증)**
- 최소 1~2개 후보 기능에 5명 중 **3명 이상**이 *"이건 정기적으로 쓸 것 같다"* 응답

**결과별 의사결정 분기**
- **Pass** → 재방문 메커니즘 자체가 비즈니스 리스크 → 한 번 들어와도 만족도 높은 SEO + 광고 모델로 단순화.
- **Fail** → 어떤 기능이 재방문을 만드는지 확인된 만큼, **그 기능을 핵심으로 키움** (예: 정비 알림 → 회원가입 → 차량 등록 → 보험·정비소 제휴 수익).

### H3. 법적 책임 — AI 챗봇 오답 리스크

**가설 진술**
> AI 챗봇이 법규를 잘못 답해서 사용자가 사고·위반에 휘말리면, **면책 조항만으로는 부족하고 실제 법적 책임이 발생할 수 있다.**

**검증 방법**
- (a) **사용자 신뢰 수준** — P1-Q7, P2-Q8에서 *"AI 챗봇 답변을 100% 신뢰하겠나"* 측정.
- (b) **법무 자문** — 변호사/로톡 등 법률 상담 플랫폼에 1회 자문. 면책 조항 + 출처 명시 + 사용자 동의 절차로 책임 회피 가능한지 확인.
- (c) **유사 서비스 벤치마킹** — 법률 챗봇(예: 로앤굿, 캐스노트 AI), 의료 챗봇 등의 면책 문구·UX 패턴 조사.

**Pass 기준 (운영 가능)**
- (b) 자문 결과 *"면책 + 출처 + 동의 절차로 회피 가능"*
- AND (a) 사용자가 *"공식 자료도 함께 보겠다"* 가 다수(보수적)

**Fail 기준 (운영 위험)**
- 자문 결과 면책으로 회피 불가
- OR 사용자가 챗봇 답변을 맹신하는 경향이 강함 (5명 중 3명 이상 *"AI 답변 그대로 따를 것 같다"*)

**결과별 의사결정 분기**
- **Pass** → 면책 + 출처 명시 + 챗봇 답변에 *"공식 자료 확인 필수"* 경고를 강하게 노출하는 정도로 운영.
- **Fail** → AI 챗봇을 *"법규 답변"* 용도에서 제외. *"운전 매너 / 차량 관리 / 사고 대처 절차"* 처럼 법적 다툼이 덜한 영역으로만 한정.

### H4. 차별점 — 유튜브·범용 LLM 대체 불가성

**가설 진술**
> 유튜브가 못 하는 *"자연어로 상황 물어보기"* 가 본 서비스의 진짜 차별점이고, 사용자는 이걸 사용하기 위해 본 서비스를 선택한다.

**검증 방법**
- (a) **자연 행동** — P1-Q3/Q4, P2-Q3/Q4, P3-Q3/Q4에서 *"평소 운전 관련 정보 어디서 찾나"* 자유 답변 → 유튜브 / 구글 / 카페 / AI 챗봇 비중 파악.
- (b) **범용 LLM 비교** — 데모 후 *"같은 질문 ChatGPT/Gemini에 한 적 있나, 어느 쪽이 더 도움될 것 같나"* 질문 (P1-Q8, P2-Q9, P3-Q8).
- (c) **지불 의향** — *"월 3,000원이면 쓸 의향 있나"* (P1-Q9, P2-Q10, P3-Q9).

**Pass 기준 (차별점 검증)**
- 5명 중 **3명 이상**이 *"운전 전용 + 출처 명시 + 한국 법규 특화"* 라는 차별점에 동의
- AND *"ChatGPT보다 이걸 쓸 것 같다"* 응답이 다수

**Fail 기준 (차별점 부족)**
- 5명 중 3명 이상이 *"ChatGPT/유튜브로 충분"* 응답
- OR *"운전 전용이라는 이유로 굳이 따로 쓸 이유는 없다"* 응답이 다수

**결과별 의사결정 분기**
- **Pass** → 챗봇을 핵심 가치로 마케팅. 한국 법규 데이터 정밀도·출처 신뢰도 강화에 리소스 집중.
- **Fail** → 챗봇을 부가 기능으로 격하. **콘텐츠 큐레이션**(유튜브 영상 + 공식 자료 정리) 또는 **계산기 중심 도구 사이트** 로 피벗 검토.

## 2. 페르소나별 인터뷰 질문지

각 질문 끝의 `[H?]` 태그는 검증 가설 매핑.

### 2.1 P1 — 예비 운전자 (면허 취득 고민·준비 중)

**워밍업**
1. 운전면허는 언제쯤 따실 계획이세요?
2. 면허 따려고 결심한 이유가 뭐예요? (출퇴근 / 가족 / 취미 / 여행 …)

**자연 행동 질문 — 데모 *전*에 묻기**
3. 면허에 대해 궁금한 게 생기면 보통 어디서 정보 찾으세요? (자유 답변 → 유튜브 / 블로그 / 카페 / 학원 문의 / AI 챗봇 …) `[H4]`
4. 가장 최근에 검색했던 면허 관련 키워드 떠올려보면 뭐였어요? 결과 만족하셨어요? `[H4]`

**가설 검증 질문**
5. (사이트 콘셉트 한 줄 설명 후) 면허 따고 나면 이런 사이트에 계속 들어올 것 같으세요? 언제까지요? `[H1]`
6. 이런 기능 중 끌리는 게 있어요? — 커뮤니티 / 사고 대처 절차 / 차량 유지비 계산기 / 매월 새 콘텐츠 알림 / 즐겨찾기 (복수 선택 가능) `[H2]`
7. AI가 알려주는 운전 법규 정보, 얼마나 믿을 것 같으세요? 100% 따를 건가요, 아니면 다시 공식 자료 확인하실 건가요? `[H3]`

**데모 (5분)**
- 메인 → 콘텐츠 1건 → 범칙금 계산기 → 챗봇 질의 1건 (*"비보호 좌회전이 뭐예요?"* 등)

**데모 후 차별점·지불의향 질문**
8. 같은 질문을 ChatGPT한테 해본 경험 있어요? 이거랑 ChatGPT 답변, 어느 쪽이 더 도움될 것 같으세요? `[H4]`
9. 만약 이 사이트가 유료(월 3,000원)라면 쓸 의향 있어요? 5,000원이면요? `[H4]`

**마무리**
10. 본인이 이 사이트 만든다면 가장 먼저 만들 기능 하나는?

### 2.2 P2 — 실주행 1년 미만 초보 운전자

**워밍업**

**자연 행동 질문**
3. 그 당황했던 순간, 해결책 어떻게 찾으셨어요? (검색? 가족·지인? 그냥 넘김?) `[H4]`
4. 운전하면서 모르는 표지판이나 상황 만났을 때 보통 어디서 정보 찾으세요? `[H4]`

**가설 검증 질문**
5. 지금 본인이 "초보운전"이라고 느끼세요? `[H1]`
6. 언제쯤 *"이제 초보 졸업이다"* 싶을 것 같으세요? 졸업한 다음에도 운전 관련 사이트에 들어올 것 같으세요? `[H1]`
7. 매주/매월 반복적으로 운전 관련해서 궁금한 거 있어요? 이런 기능이 있으면 정기적으로 들어오실 것 같아요? — 정비 시기 알림 / 사고 사례 정리 / Q&A 커뮤니티 / 계산기 즐겨찾기 `[H2]`
8. AI 챗봇이 알려준 법규대로 했는데 단속 걸리면 어떨 것 같으세요? AI 답변, 100% 믿으실래요? `[H3]`

**데모**
- 메인 → 콘텐츠 1건 → 챗봇 질의 (*"좁은 골목에서 차랑 마주쳤어요"* 등 P2 시나리오)

**데모 후 차별점·지불의향 질문**
9. 평소 운전 관련 정보 찾을 때 유튜브 자주 쓰세요? 유튜브 대신 이거 쓰실 것 같아요? `[H4]`
10. 월 3,000원이라면 가입할 의향? 5,000원이면요? `[H4]`

**마무리**
11. 본인 친구 초보운전자한테 이 사이트 추천한다면 한 줄로 뭐라고 소개하실 거예요?

**가설 검증 질문**

8. 차 관련 문제 생기면 보통 유튜브 검색하시는 편이에요? 이거랑 비교하면 어떠세요? `[H4]`
